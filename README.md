# genpdf-json-bin

Fast PDF generator

To create use:

```cargo build --release```


Example of use from the console:

``` genpdf-json-bin  file.json  file.pdf ```

``` genpdf-json-bin  base.db  file.pdf ```

or

``` genpdf-json-bin  -j file.json  -o file.pdf ```

``` genpdf-json-bin  -s base.db  -o file.pdf ```
